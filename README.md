# Hotel-Reservation-System Java,MySQL,JDBC
A simple console-based Hotel Room Reservation System built using Java and MySQL.
This project allows hotel staff to perform room bookings, view reservations, update details, delete entries, and fetch room numbers using guest information.

 Features:
✔ Reserve a Room
 =>Add guest name, room number, and contact number
 =>Checks if room is already booked
 =>Inserts new reservation if available

✔ View All Reservations
 Displays a formatted table showing:
 =>Reservation ID
 =>Guest Name
 =>Room Number
 =>Contact Number
 =>Booking Date

✔ Get Room Number
 Retrieve room number by entering:
 =>Reservation ID
 =>Guest Name

✔ Update Reservation
 Modify existing reservation details:
 =>Guest Name
 =>Room Number
 =>Contact Number

✔ Delete Reservation
 =>Remove reservation from the database using reservation ID.

✔ Exit
 =>Gracefully exits the system with a countdown animation.
